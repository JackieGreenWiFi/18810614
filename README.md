[Chromium OS＆LMSの活用方法]

（１）あらすじ
発展途上国ではコンピューターの普及が先進国ほどではありません。
（一方で先進国に比べ発展途上国の方がスマートフォンの普及率高い）

そこで、カンボジアを選びました。カンボジアのコンピューター普及率、そしてコンピューター利用者が少ない原因として大きく分けて２つあります。

１つは、パソコンが（日本で買う価格よりも）高いといことです。

そして、２つめは、コンピューター教育が普及していない、していない原因として基本的な操作ができないことなど。があります。

そこで、提案があります。
１つめの価格の問題の解決案として、
Raspberry PiやChromium OSを使うということです。

２つめの、コンピューター教育の問題の解決案として、LMSを使うということです。

（２）目的
・発展途上国（カンボジア）のコンピューターの普及並びにコンピューター教育の普及より利用者の増加

・internetの使えない地域でIT教育を可能とする

（３）問題点
　問題は大きく分けて２つあります。
　・価格の問題
　・コンピューター教育の普及の問題

※価格の問題とは、カンボジアでは日本よりもPCが高かったです。コンピューター教育の普及の問題とは、基本的な使い方やプレゼンテーションソフトの普及の問題などがあり、手に触れることのできる人が少ないということです。


（４）解決案
・価格の問題の解決案
Raspberry Piでパソコンを作ることやChromium OSをRaspberry Piにインストールして使うことで価格の問題は解決できると考えている。

※低い価格でGETできることにより、幅広い人への普及につながると考えています。そして、学校でRaspberry PiやChromium OSを使うことにより、これらのOSの普及につながると考えています。

・コンピューター教育の普及の問題の解決案

Raspberry Pi上でLMSによりインターネットにアクセスできない地域でもLMSを使うことができる、そして持ち運びもできる。LMSより、基本的な使い方などは動画によって学ぶことができる。そして、Chromium OSを使うことでプレゼンテーションのスライドなどを作る教育にもつながる。


（５）その他
カンボジアでこのプロジェクトが実現したら、ラオスやインドネシア、それからアフリカなどの、まだコンピューターが普及していない地域の教育支援につながる、そしてコンピューターの利用者の普及にもつながる。
